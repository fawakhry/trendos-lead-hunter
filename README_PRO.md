# صياد العملاء Pro — نسخة مستقلة آمنة

نسخة مستقلة تعمل من GitHub Pages بدون TrendOS وبدون Backend إلزامي.

## أهم الإمكانيات

- إضافة لينكات جروبات/صفحات/بحث فيسبوك.
- تحديد مدة مراجعة لكل مصدر: 10، 15، 30 دقيقة أو أكثر.
- Safe Auto Mode: طابور بحث يفتح روابط بحث فيسبوك الجاهزة حسب الجدول.
- تدوير كلمات البحث لكل مصدر.
- كلمات خاصة لكل مصدر.
- فتح بحث واحد أو 3 أبحاث دفعة واحدة.
- تنبيهات المتصفح وصوت تنبيه.
- حفظ فرصة يدويًا من النص والرابط.
- تحليل محلي للنص بدون API.
- ردود مقترحة باسم مطبعجي بنها.
- منع تكرار الفرص بنفس الرابط أو نفس النص.
- شاشة الفرص الساخنة والمتابعات.
- زر متابعة بعد ساعة.
- تصدير CSV ونسخة احتياطية JSON.
- إضافة Chrome Pro لحفظ النص المحدد وفتح البرنامج.

## تنبيه مهم

النسخة لا تستخدم Scraper ولا تسحب بوستات أو كومنتات الجروبات تلقائيًا. هي تفتح لك بحث فيسبوك الجاهز حسب الجدول، وأنت تحفظ النص/الرابط من خلال الإضافة أو النموذج.

## الرفع على GitHub Pages

ارفع محتوى المجلد كله في جذر الريبو:

```text
index.html
styles.css
app.js
config.js
chrome-extension/
preview/
README_PRO.md
```

ثم فعل GitHub Pages من:

```text
Settings -> Pages -> Deploy from branch -> main -> root
```

## تشغيل إضافة Chrome

1. افتح `chrome://extensions`.
2. فعل Developer mode.
3. اضغط Load unpacked.
4. اختر مجلد `chrome-extension`.
5. ضع رابط GitHub Pages الخاص بالبرنامج داخل الإضافة.

## طريقة الاستخدام اليومية

1. افتح البرنامج.
2. أضف مصادر الجروبات والصفحات.
3. حدد لكل مصدر 10 أو 15 دقيقة.
4. فعل تشغيل المراقبة.
5. فعل فتح تلقائي حسب رغبتك.
6. عندما يفتح البحث، اختار بوست/كومنت مناسب.
7. حدد النص واستخدم إضافة Chrome لفتحه داخل صياد العملاء.
8. احفظ الفرصة وانسخ الرد المقترح.
