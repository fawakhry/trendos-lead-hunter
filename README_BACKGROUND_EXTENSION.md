# Lead Hunter Pro Background Scheduler

هذه النسخة تضيف تشغيل شبه أوتوماتيك آمن عبر Chrome Extension:

- جدولة جروبات وصفحات كل 10/15/30 دقيقة.
- فتح بحث فيسبوك تلقائيًا في تبويب منفصل عند ميعاد المصدر.
- تدوير كلمات البحث لكل مصدر.
- إشعارات متصفح عند فتح بحث جديد.
- Badge على الإضافة بعدد المصادر المستحقة.
- زر عائم داخل صفحات فيسبوك: “🎯 احفظ فرصة”.
- حفظ النص المحدد واللينك داخل برنامج صياد العملاء.

## التشغيل

1. ارفع ملفات البرنامج على GitHub Pages.
2. افتح Chrome ثم: chrome://extensions
3. فعّل Developer mode.
4. اضغط Load unpacked.
5. اختر مجلد chrome-extension.
6. افتح الإضافة واكتب رابط GitHub Pages الخاص بالبرنامج.
7. أضف لينكات الجروبات والكلمات وفعّل “تشغيل الخلفية”.

## المهم

النسخة لا تسحب بوستات أو كومنتات فيسبوك تلقائيًا ولا تعمل scraping. هي تفتح البحث في مواعيده، وأنت تحفظ الفرصة بالزر العائم أو من popup الإضافة.

## تحديث إصلاح 2026-07-10

- إصلاح زر إلغاء في نافذة إضافة مصدر حتى يغلق النافذة بدون طلب ملء الحقول.
- إضافة زر حذف للمصادر داخل كروت المصادر المستحقة.
- إضافة أزرار حذف/إيقاف/تفعيل داخل لوحة Lead Hunter Pro.
- حذف المصدر لا يحذف الفرص القديمة المحفوظة، فقط يزيل المصدر من جدول المراقبة.
